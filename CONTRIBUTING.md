# Contributing to OrbitalSkyShield

Thank you for your interest in contributing to OrbitalSkyShield! Even though this is an independent research project, we welcome community feedback and improvements.

## ü§ù How to Contribute

### 1. Reporting Bugs
If you find a bug, please create a new issue with:
*   **Description:** clearly explain what happened.
*   **Reproduction:** steps to reproduce the error.
*   **Environment:** Python version, OS, etc.
*   **Logs:** Paste relevant error messages.

### 2. Suggesting Enhancements
Have an idea? Open an issue tagged as `enhancement`.
*   Explain the goal.
*   Suggest an implementation approach if possible.

### 3. Pull Requests (Code Contributions)
1.  **Fork** the repository.
2.  Create a new branch: `git checkout -b feature/AmazingFeature`.
3.  **Code Standards:**
    *   Follow PEP 8 for Python.
    *   Add comments for complex logic.
    *   Ensure existing tests pass (if any).
4.  **Commit** your changes: `git commit -m 'Add some AmazingFeature'`.
5.  **Push** to the branch: `git push origin feature/AmazingFeature`.
6.  Open a **Pull Request**.
